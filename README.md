**Recallr** is a high-performance, SSR-ready flashcard ecosystem designed to bridge the gap between cognitive science and modern web design. Built for students who value speed, aesthetics, and long-term retention.



## 🚀 The Core Experience
Recallr isn't just a directory of cards; it's a second brain. Using a custom-built **Spaced Repetition System (SRS)**, the app calculates precisely when you are about to forget a concept and surfaces it for review, ensuring maximum efficiency with minimum effort.

### ✨ Key Features
- **⚡ SSR Optimized:** Lightning-fast page loads and SEO-ready architecture using Next.js.
- **🎨 Tactile UI:** Fluid, gesture-driven animations powered by Framer Motion.
- **📁 Smart Workspace:** Organise your knowledge with nested folders, tags, and intelligent search.
- **🔥 Daily Streaks:** Visual progress tracking to keep your consistency high.
- **🤖 AI Synthesis:** (Coming Soon) Transform raw lecture notes into polished decks instantly.



## 🛠️ Tech Stack
- **Frontend:** Next.js 14, TypeScript, Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Backend:** Django REST Framework (DRF)
- **Database:** PostgreSQL
